In Wix: Settings → Design → Fonts

- **Headings / Logo:** Open Sans, Weight 600 (SemiBold)
- **"crane" in logo:** Open Sans, Weight 300 (Light)
- **Body text:** Open Sans, Weight 400 (Regular)
- **Technical labels / codes:** Share Tech Mono (add via Google Fonts in Wix)

Open Sans is already in the Wix font library. Search "Open Sans".

### Logo
Upload `craneiq-wordmark-dark.svg` to Wix Media.
- Use on dark backgrounds (nav, footer, dark sections)
- Use `craneiq-wordmark-light.svg` on light backgrounds
- Use `craneiq-monogram.svg` as favicon (Wix Settings → Favicon)

### PRODUCT: craneIQ Core
**Code:** CIQ-COR
**Accent colour:** #008B9A
**Tagline:** The intelligent hoist & trolley controller

**Description:**
craneIQ Core is a hoist and trolley controller with built-in load protection functions — slack rope, shock load, and side pull detection. It includes condition monitoring with a load spectrum recorder and Design Working Period calculator. Anti-sway control is available for travel movements. Fits any overhead crane, new or existing.

**Key features:**
- Slack rope detection and protection
- Shock load protection
- Side pull detection
- Load spectrum recorder
- Design Working Period (DWP) calculator
- Anti-sway control for travel movements
- Tandem hoist monitoring support
- New and retrofit installations

**Typical applications:** Steel stockholders · Automotive manufacturing · Waste & recycling · General manufacturing · Warehousing & logistics

**Download datasheet:** craneiq-core-datasheet.pdf

---

### PRODUCT: craneIQ Automation
**Code:** CIQ-AUT
**Accent colour:** #4488CC
**Tagline:** Recipe-based plug-and-play crane automation

**Description:**
craneIQ Automation automates crane systems using a standard recipe-based system. Operators teach movements by driving the crane — the system records the path and replays it on command, like robot teach programming but for overhead cranes. No specialist engineers or complex programming required. Includes built-in anti-sway and condition monitoring.

**Key features:**
- Recipe-based automation system
- Teach and replay programming
- No specialist engineering required
- Built-in anti-sway for travel
- Condition monitoring included
- Compatible with any crane control system
- Eliminates project programming time
- New and retrofit installations

**Typical applications:** Automated warehousing · Repetitive process handling · Coil & reel handling · Steel & aluminium processing · Goods-in / goods-out automation

**Download datasheet:** craneiq-automation-datasheet.pdf

### PRODUCT: craneIQ Monitoring
**Code:** CIQ-MON
**Accent colour:** #8855DD
**Tagline:** Standalone crane health monitoring

**Description:**
craneIQ Monitoring is a standalone condition monitoring system that can be added to any overhead crane without affecting its existing control system. It continuously records load spectrum data, calculates Design Working Period consumption, logs hours run and number of operations, and monitors vibration. All data is stored locally and can be exported to craneIQ Cloud.

**Key features:**
- Load spectrum recorder
- DWP calculator and tracker
- Hours run logging
- Number of operations counter
- Vibration monitoring
- Historical data trending
- 32 GB local data storage
- Standalone — no control system changes required

**Typical applications:** Existing cranes without monitoring · Compliance & insurance requirements · Planned maintenance programmes · High-duty cycle applications · Asset life management

**Download datasheet:** craneiq-monitoring-datasheet.pdf

### PRODUCT: craneIQ Vision
**Code:** CIQ-VIS
**Accent colour:** #009966
**Tagline:** Hook tracking and snag prevention

**Description:**
craneIQ Vision is a camera-based hook tracking system that detects and prevents side pull and snag events in real time. The system continuously monitors the position of the hook relative to the load and intervenes before a damaging lift can occur. An auto-centre function can automatically position the hook directly above the load before the hoist is operated, significantly reducing operator errors and crane damage.

**Key features:**
- Real-time hook position tracking
- Side pull detection and prevention
- Snag event detection and prevention
- Automatic hook centring over load
- Operator safety alerts and warnings
- Works with any hoist type
- IP67 rated camera unit
- New and retrofit installations

**Typical applications:** Steel coil and slab handling · Container and pallet handling · Any load where snag risk exists · High-value load handling · Operator training environments

**Download datasheet:** craneiq-vision-datasheet.pdf

### PRODUCT: craneIQ Antisway
**Code:** CIQ-ASW
**Accent colour:** #CC7700
**Tagline:** Retrofit anti-sway — no special sensors required

**Description:**
craneIQ Antisway is an anti-sway system that can be retrofitted to any overhead crane that has Variable Frequency Drives installed on the travel motions. No special sensors, encoders, or cameras are required. The open-loop algorithm profiles the VFD speed references to minimise load swing. Works on any number of axes and commissions in less than one day.

**Key features:**
- No special sensors or encoders required
- Works with existing VFDs only
- Single, dual, or triple axis
- Significantly reduces load swing
- Improves operator productivity and cycle time
- Commissions in less than one day
- Compatible with all major VFD brands
- New and retrofit installations

**Typical applications:** High bay warehousing · Steel and coil handling · Shipyard and offshore · Any crane with travel VFDs · Productivity improvement projects

**Download datasheet:** craneiq-antisway-datasheet.pdf

---

### PRODUCT: craneIQ Zoning
**Code:** CIQ-ZON
**Accent colour:** #CC3344
**Tagline:** No-fly zones and collision prevention

**Description:**
craneIQ Zoning prevents overhead cranes entering defined no-fly zones or colliding with buildings, other cranes, floor-level machinery, or personnel areas. Zone boundaries are configured via software and the system automatically decelerates and stops the crane before it can enter a restricted area. Works at both crane rail level and floor level.

**Key features:**
- Configurable no-fly zone boundaries
- Multi-crane anti-collision
- Floor-level machine and personnel protection
- Crane-level building structure protection
- Automatic deceleration and stop
- Visual and audible warnings
- Unlimited configurable zones
- New and retrofit installations

**Typical applications:** Multi-crane bays · Cranes over pedestrian areas · Restricted process zones · Buildings with structural limits · Cranes sharing runway with other machines

**Download datasheet:** craneiq-zoning-datasheet.pdf

### PRODUCT: craneIQ LoadTrack
**Code:** CIQ-LTK
**Accent colour:** #CC9900
**Tagline:** Load logging and wireless weight display

**Description:**
craneIQ LoadTrack logs every load lifted by the crane and displays the live weight on wireless display units that can be positioned anywhere within the factory. It also logs process cycles driven by the crane — making it ideal for galvanising plants, waste-to-energy facilities, or any process where crane lift data drives production records or billing.

**Key features:**
- Continuous load logging for every lift
- Wireless weight display receivers
- Displays locate anywhere within 100m
- Process cycle logging and reporting
- Galvanising plant compatible
- Waste-to-energy cycle tracking
- Historical lift reports and CSV export
- Modbus and cloud data output

**Typical applications:** Galvanising plants · Waste-to-energy facilities · Scrap metal handling · Process manufacturing · Any application requiring lift records or process billing

**Download datasheet:** craneiq-loadtrack-datasheet.pdf
