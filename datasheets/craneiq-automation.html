Overview

Automating an overhead crane has traditionally meant expensive PLC programming projects, specialist commissioning engineers, lengthy validation periods, and months of disruption before the system works reliably. craneIQ Automation changes all of that. Using the same teach-and-replay principle as robot programming, operators simply drive the crane through the movements they want to automate — the system records the path and replays it on demand at the press of a button. No code is written. No specialist engineer is needed. Human error is removed from repetitive precise movements. The result is a fully automated crane cycle that any operator can run, commissioned in a day, that improves safety, increases throughput and frees your most experienced operators for higher value tasks. Anti-sway and condition monitoring are built in as standard — no additional hardware required.

01 — Teach
Drive the crane
An operator drives the crane through the required movement — pick position, travel path, place position. The system records every axis precisely. No programming. No code. No engineer.
02 — Store
Save the recipe
The recorded movement is saved as a named recipe. Store as many routines as you need — different loads, different processes, different destinations. Switch between them instantly.
03 — Replay
Repeat on demand
Any operator selects the recipe and initiates the cycle. The crane executes the movement precisely — with built-in anti-sway — every time. No experience required. No variation between operators.
