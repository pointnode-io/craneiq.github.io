## Products

- craneIQ Core — Condition monitoring & hoist controller
- craneIQ Automation — Teach-and-replay crane automation  
- craneIQ Monitoring — Standalone crane health monitoring
- craneIQ Cloud — Remote visibility & alerts
- craneIQ Vision — Hook tracking & snag prevention
- craneIQ Antisway — Anti-sway, no sensors required
- craneIQ Zoning — No-fly zones & collision prevention
- craneIQ LoadTrack — Load logging & wireless display
