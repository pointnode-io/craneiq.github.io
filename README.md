## About

craneIQ is a product of Pointnode Ltd, West Midlands, UK.

Eight products for overhead crane control, automation, monitoring and safety:
- craneIQ Core
- craneIQ Automation
- craneIQ Monitoring
- craneIQ Cloud
- craneIQ Vision
- craneIQ Antisway
- craneIQ Zoning
- craneIQ LoadTrack
